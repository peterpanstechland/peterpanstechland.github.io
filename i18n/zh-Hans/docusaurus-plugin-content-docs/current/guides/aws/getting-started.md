# AWS 入门

面向初学者的 Amazon Web Services 入门指南，专注于 IoT 和 AI 项目最相关的服务。

## 为什么选择 AWS？

AWS 提供全面的云服务套件：

- **IoT 连接** - 安全连接数百万设备
- **无服务器计算** - 无需管理服务器即可运行代码
- **AI/ML 服务** - 无需 ML 专业知识即可访问强大模型
- **全球规模** - 低延迟全球部署

## 创客必备服务

### IoT 服务
- **AWS IoT Core** - 设备连接和消息路由
- **AWS Greengrass** - 边缘计算和本地执行
- **AWS IoT Analytics** - 数据分析和洞察

### 计算
- **Lambda** - 无服务器函数
- **EC2** - 虚拟服务器

### AI/ML
- **Amazon Bedrock** - 基础模型 API 访问
- **SageMaker** - ML 模型训练和部署

### 存储
- **S3** - 对象存储
- **DynamoDB** - NoSQL 数据库

## 前置条件

开始使用 AWS 之前：

1. 创建 [AWS 账户](https://aws.amazon.com)
2. 设置具有适当权限的 IAM 用户
3. 安装 AWS CLI
4. 本地配置凭证
