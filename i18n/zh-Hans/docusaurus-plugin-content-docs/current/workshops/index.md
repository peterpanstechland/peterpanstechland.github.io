# 工作坊

为社区活动、聚会和自学设计的动手实践材料。每个工作坊包含完整的说明、代码示例和资源。

## 工作坊格式

每个工作坊的结构：

1. **介绍** (15 分钟) - 概念和目标
2. **环境准备** (15 分钟) - 环境和前置条件
3. **动手实践** (60-90 分钟) - 分步实现
4. **问答环节** (15 分钟) - 讨论和故障排除
