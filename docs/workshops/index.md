# Workshops

Hands-on workshop materials designed for community events, meetups, and self-paced learning. Each workshop includes complete instructions, code samples, and resources.

## Workshop Format

Each workshop is structured as:

1. **Introduction** (15 min) - Concepts and objectives
2. **Setup** (15 min) - Environment and prerequisites
3. **Hands-on** (60-90 min) - Step-by-step implementation
4. **Q&A** (15 min) - Discussion and troubleshooting
