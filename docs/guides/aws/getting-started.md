# Getting Started with AWS

A beginner-friendly introduction to Amazon Web Services, focused on the services most relevant to IoT and AI projects.

## Why AWS?

AWS provides a comprehensive suite of cloud services that enable:

- **IoT Connectivity** - Connect millions of devices securely
- **Serverless Computing** - Run code without managing servers
- **AI/ML Services** - Access powerful models without ML expertise
- **Global Scale** - Deploy worldwide with low latency

## Key Services for Makers

### IoT Services
- **AWS IoT Core** - Device connectivity and message routing
- **AWS Greengrass** - Edge computing and local execution
- **AWS IoT Analytics** - Data analysis and insights

### Compute
- **Lambda** - Serverless functions
- **EC2** - Virtual servers

### AI/ML
- **Amazon Bedrock** - Foundation model API access
- **SageMaker** - ML model training and deployment

### Storage
- **S3** - Object storage
- **DynamoDB** - NoSQL database

## Prerequisites

Before starting with AWS:

1. Create an [AWS Account](https://aws.amazon.com)
2. Set up IAM user with appropriate permissions
3. Install AWS CLI
4. Configure credentials locally
